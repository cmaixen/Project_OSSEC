\begin{abstract}

Malware is a know threat to many computer users for several years. This report will focus on a special kind of malware, namely ransomeware. Research will be done on the spreading, used vulnerabilities and evolution generation of the ransomeware. At last will we implement a ransomeware, based on the findings. This self-written ransomeware will have all the main features as a ransomeware used for malicious practices in real life.  

\end{abstract}
